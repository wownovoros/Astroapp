# AstroApp — Инструкция пользователя

## Что это за приложение

AstroApp — Telegram Mini App с двумя разделами:
1. **Гороскоп** — персональный прогноз по знаку зодиака на сегодня и на неделю
2. **Натальная карта** — полный астрологический расклад по дате и времени рождения

---

## Как открыть приложение

1. Найди бота в Telegram (по ссылке или через поиск)
2. Нажми `/start`
3. Появится кнопка **«Открыть Astro ✨»** — нажми её
4. Приложение откроется прямо внутри Telegram

---

## Раздел: Гороскоп

1. На главной нажми **«Гороскоп»**
2. Выбери свой знак зодиака из сетки (12 знаков с датами)
3. Откроется страница с двумя вкладками:
   - **«На сегодня»** — ежедневный прогноз (меняется каждый день)
   - **«На неделю»** — недельный прогноз (меняется каждую неделю)
4. Тексты загружаются автоматически

---

## Раздел: Натальная карта

1. На главной нажми **«Натальная карта»**
2. Введи дату рождения в формате **ДД-ММ-ГГГГ** (например: 15-06-1990)
3. Введи время рождения в формате **ЧЧ:ММ** (например: 14:30)
   - Если время неизвестно — оставь 12:00 (точность карты будет ниже)
4. Нажми **«Рассчитать натальную карту»**
5. Откроется карта с четырьмя вкладками:

### Вкладки натальной карты

| Вкладка | Что показывает |
|---------|---------------|
| **Карта** | Визуальный круг с зодиакальными секторами, планетами, аспектами, ASC и MC |
| **Планеты** | Таблица: планета, знак, градус, признак ретроградности |
| **Аспекты** | Углы между планетами: зелёные (гармоничные) и красные (напряжённые) |
| **Разбор** | Подробная интерпретация каждой планеты в своём знаке |

---

## Что рассчитывает карта

- **12 планет**: Солнце, Луна, Меркурий, Венера, Марс, Юпитер, Сатурн, Уран, Нептун, Плутон, Хирон, Северный Узел
- **Асцендент (ASC)** и **Середина Неба (MC)**
- **Ретроградность** планет
- **32+ аспекта** между планетами (соединение, тригон, секстиль, квадрат, оппозиция, квинконс)
- Подробные интерпретации для каждой планеты в знаке

## Где хранится и как деплоится

| Компонент | Где |
|-----------|-----|
| Код проекта | GitHub репозиторий |
| Сервер | Render.com (бесплатный тариф) |
| Бот | Telegram Bot API (webhook) |
| URL приложения | https://astroapp-gk0z.onrender.com/ |

**Деплой автоматический**: любой `git push` в ветку `main` → Render автоматически пересобирает и запускает новую версию.

---

## Как обновить приложение (сделать изменения)

1. Попроси ассистента внести правки в файлы
2. После изменений запусти в PowerShell:
   ```powershell
   cd "D:\cursor обучение\.cursor\astro_app"
   Remove-Item ".git\index.lock" -Force -ErrorAction SilentlyContinue
   git add .
   git commit -m "описание изменений"
   git push origin main
   ```
3. Подожди 2–3 минуты — Render задеплоит автоматически

## Если что-то сломалось

| Проблема | Решение |
|----------|---------|
| Бот не отвечает на /start | Открой https://astroapp-gk0z.onrender.com/setup в браузере |
| Гороскоп не загружается | Это нормально при первом обращении — сервер «просыпается» ~30 сек |
| Ошибка в натальной карте | Проверь формат даты: ДД-ММ-ГГГГ |
| MINI_APP_URL не настроен | В Render → Environment → добавь переменную MINI_APP_URL |

---

## Переменные окружения (Render Dashboard → Environment)

| Переменная | Описание | Пример |
|------------|----------|--------|
| `BOT_TOKEN` | Токен бота от @BotFather | 123456:ABC... |
| `WEBHOOK_SECRET` | Секрет для вебхука | astrosecret123 |
| `MINI_APP_URL` | Публичный URL приложения | https://astroapp-gk0z.onrender.com |
| `DEFAULT_LAT` | Широта по умолчанию | 55.7558 (Москва) |
| `DEFAULT_LON` | Долгота по умолчанию | 37.6176 (Москва) |
| `DEFAULT_TZ_OFFSET` | Часовой пояс | +03:00 |
